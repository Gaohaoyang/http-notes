Introduction

使用 HTTP 协议访问 Web

在浏览器地址栏输入 URL 后，信息会被送往某处。然后从某处得到回复，内容就会显示在 Web 页面上。

Web 使用一种名为 HTTP（HyperText Transfer Protocol，超文本传输协议）的协议作为规范。

可以说 Web 是简历在 HTTP 协议上通信的。
